# Specification

## Goal

Maintain a public GitHub profile README that summarizes public work, public demos, and public repositories without exposing private planning details or personal contact details.

## Non-Goals

- Publish private project plans, private workspaces, unreleased roadmap details, credentials, or personal contact details.
- Claim employment affiliation, client relationship, certification, production readiness, or regulated deployment readiness.
- Use the profile README as a complete resume or private business plan.

## Public Boundary

This repository is a public profile surface. It may link to public repositories and public demos. It should use the GitHub handle as the public identity boundary.

## Success Criteria

- The profile explains public work clearly.
- Public links are intentional and safe.
- Pre-release checks pass before public updates.
